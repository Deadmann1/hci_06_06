Negative Findings (Problems)
-----------------

o MS-Mob-Neg01

  Title:  Support Chat Icon

  Description:

  The icon for the support chat cannot be minimized or removed.
  In some cases this is annoying.
  
  Video Clip: ms-mob-neg01-support-chat-icon.mp4

  Heuristic: A08 Aesthetic and Minimalist Design

  How Reproducible?:  Every page
  
  
  
  
o MS-Mob-Neg02

  Title:  No incremental search

  Description:

  The search field does not provide an incrementeal search feature.
  This kind of feature is very common, especially for mobile friendly websites.
  
  Video Clip: ms-mob-neg02-incremental-search.mp4

  Heuristic: A07 Flexibility and Efficiency of Use

  How Reproducible?:  Every page -> use search function next to menu icon
  
  
  
  
o MS-Mob-Neg03

  Title:   Inconsistency on collapsed menu

  Description:

  When opening the menu and one of the categories e.g. "Anreise & Verkehr" we get a list of collapsed menus.
  If we click on the text of the collapsed menus instead of expanding we get redirected on a new page.
  If we click on "Unsere Tipps" though the menu expands.
  
  Video Clip: ms-mob-neg03-inconsistent-menu.mp4

  Heuristic: A04 Consistency

  How Reproducible?:  Homepage -> Any category (e.g. "Essen & Trinken") -> Try to click on text of "Unsere Tipps" or other menu

  Title:  Tiles crowd screen

  Description:

  Video Clip: ms-mob-neg04-crowded-tiles.mp4
 
  Heuristic: A08 Aesthetic and Minimalist Design

  How Reproducible?:  Homepage -> "Shopping & Lifestyle" -> "Einkaufsviertel"

o MS-Mob-Neg08 [corresponds to MS-PC-Neg07]

  Title: Interactive Map Auto Zoom

  Description:

  The website offers an interactive map with points of interest that can be filtered by categories using buttons on the map (Sightseeing, Museen, Hotels, etc.).
  After changing the filter the map automatically zooms out.
  This is very annoying when looking for a hotel, etc. next to a certain poi.

  Video Clip:  ms-mob-neg08-auto-zoom.mp4

  Heuristic:  A04 Consistency

  How Reproducible?:  Homepage -> "Anreise & Verkehr" -> "Interaktiver Stadtplan" -> zoom anywhere -> activate a filter
  
  
  
  
o MS-Mob-Neg09 [corresponds to MS-PC-Neg08]

  Title: Hotel map extreme zoom

  Description:

  If we open the page for a certain hotel and open its provided map we get a nice overview where the hotel is located.
  Should we close the map and open it again we get the same map again but now its extremely zoomed in to where it is practically useless.
  The user has to manually zoom out to see anything of the surroundings again.

  Video Clip:  ms-mob-neg09-hotel-zoom.mp4

  Heuristic:  A04 Consistency

  How Reproducible?:  Homepage -> "Hotels & Appartements" -> "Hotels" -> select any hotel -> open the map -> close the map -> open the map
